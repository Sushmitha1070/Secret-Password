# Secret-Password
Overview: This project is a web application that reveals a secret text only when the correct password is entered. It uses a combination of HTML, CSS, JavaScript on the front end, and Node.js with Express on the back end.
Features:

Password Input: Users can enter a password in an input field.
Form Submission: The password is submitted to the server for verification.
Server Verification: The server checks if the submitted password matches the predefined correct password.
Secret Text Display: If the password is correct, the server responds with the secret text, which is then displayed to the user.
